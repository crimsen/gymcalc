Ausgabe von Wettkampflisten

In der Vorbereitung eines Wettkampfes sind unterschiedliche Listen auszudrucken. Dazu gehören Riegenlisten, die veröffentlicht werden sollten, damit die Starter wissen in welche Riege sie eingeteilt sind und in welcher Reihenfolge die einzelnen Geräte bzw. Stationen absolviert werden. Weiterhin sind Wertungslisten für die Kampfrichter notwendig und ggf. Listen als Meldebestätigung oder zur Abrechnung von Startgeldern.

GymCalc bietet eine sehr flexible Möglichkeit diese Listen auszugeben. Diese Listen basieren allesamt auf "Schablonen" in denen angegeben ist was und wie es ausgegeben werden soll. Hier in der Einführung soll sich auf die Ausgabe von vorgefertigten Schablonen beschränkt werden. Will man die Schablonen anpassen oder eigene ausgefeiltere, zusätzliche, bessere Schablonen anlegen, so sei auf die Abschnitt Schablonen verwiesen.

Procedure 15. Schablonen vorbereiten

Die Schablonen sind zur Zeit nicht Teil von GymCalc. Das ist so gemacht, damit die Schablonen auch leicht geändert werden können. Aus diesem Grund müssen sie aus dem Internet geladen werden und in einem eigenen Verzeichnis abgelegt werden.

  1. Lade vorgefertigte Schablonen von http://www.gymcalc.de/schablonen und lege die Schablonen in einem Verzeichis (z.B. Eigene Dokumente\GymCalc Schablonen) ab. In diesem Verzeichniss sollten sich dann die Schablonen:

    • registration-organization.odt für die Meldeübersicht / Meldebestätigung
    • chain.odt für die Riegenliste
    • juryform.odt für die Wertungslisten
    • jury.odt für die Kampfrichterliste

    vorhanden sein.

  2. Das Verzeichniss, in dem sich die Schablonen befinden, muss GymCalc bekannt gemacht werden.

    1. Drücke dazu auf Window → Preferences

    2. Wähle auf der linken Seite Schablonen

    3. Wähle unter Template Directory das Verzeichniss in dem sich die Schablonen befinden.

    4. Wähle unter Generation Directory ein Verzeichniss das temporäre Dateien enthält. Dieses Verzeichnis wird von GymCalc verwendet um teilweise ausgefüllte Schablonen zwischenzuspeichern. Das Verzeichniss sollte somit beschreibbar sein. Die dort angelegten Dateien können aber wieder getrost gelöscht werden.

Procedure 16.  Ausgabe einer Meldeübersicht / Meldebestätigung

  1. Öffne den "Übersicht"-Editor.

  2. Wähle aus dem Menu File → Mit Vorlage Drucken...

  3. Wähle als Vorlage registration-organization.odt nur drückeNext >

  4. Wähle ein Verzeichniss und gib einen Dateinamen ein, unter dem die Meldeübersicht bzw. Meldebestätigung abgelegt werden soll. DrückeFinish.

  5. Ergebnis ist eine Datei mit ausgewähltem Namen im ausgewählten Verzeichnis. Diese Datei ist im "ODT"-Format, welches mit einem der Schreibprogramme wie "LibreOffice" geöffnet werden kann.

Procedure 17.  Ausgabe der Riegenliste

  1. Öffne den "Übersicht"-Editor.

  2. Wähle aus dem Menu File → Mit Vorlage Drucken...

  3. Wähle als Vorlage chain.odt nur drückeNext >

  4. Wähle ein Verzeichniss und gib einen Dateinamen ein, unter dem die Riegenliste abgelegt werden soll. DrückeFinish.

  5. Ergebnis ist eine Datei mit ausgewähltem Namen im ausgewählten Verzeichnis. Diese Datei ist im "ODT"-Format, welches mit einem der Schreibprogramme wie "LibreOffice" geöffnet werden kann.

Procedure 18.  Ausgabe der Wertungslisten

  1. Öffne den "Übersicht"-Editor.

  2. Wähle aus dem Menu File → Mit Vorlage Drucken...

  3. Wähle als Vorlage juryform.odt nur drückeNext >

  4. Wähle ein Verzeichniss und gib einen Dateinamen ein, unter dem die Wertungslsiten abgelegt werden soll. DrückeFinish.

  5. Ergebnis ist eine Datei mit ausgewähltem Namen im ausgewählten Verzeichnis. Diese Datei ist im "ODT"-Format, welches mit einem der Schreibprogramme wie "LibreOffice" geöffnet werden kann.

    Note

    Die Vorlage für die Wertungslisten ist so angelegt, dass jede Riege für jedes Gerät ein A5-Blatt erhält. Wird also auf A4-Blättern (wie üblich) gedruckt, so kann man mit LibreOffice zunächst die erste Hälfte der Seiten ausdrucken, das Papier nochmals aber anders herum einlegen und dann die zweite Hälfte der Seiten drucken.

Procedure 19.  Ausgabe der Kampfrichterliste

  1. Öffne den "Übersicht"-Editor.

  2. Wähle aus dem Menu File → Mit Vorlage Drucken...

  3. Wähle als Vorlage jury.odt nur drückeNext >

  4. Verändere in der Parameterseite nichts und drückeNext >

  5. Wähle ein Verzeichniss und gib einen Dateinamen ein, unter dem die Kampfrichterliste abgelegt werden soll. DrückeFinish.

  6. Ergebnis ist eine Datei mit ausgewähltem Namen im ausgewählten Verzeichnis. Diese Datei ist im "ODT"-Format, welches mit einem der Schreibprogramme wie "LibreOffice" geöffnet werden kann.
